### 基础查询

```mysql
语法：
select 查询列表
from 表明

查询列表可以是：表中的字段、常量值、表达式、函数
查询结果是一个虚拟的表格

查询表中的单个字段
SELECT last_name FROM employees;
查询表中的多个直段
SELECT last_name,email,salary FROM employees;
查询表中的所有字段
SELECT * FROM employees; // * 表中字段顺序与原始表一模一样
表头先写 use 对于库名；
··  1旁边的着重号 可以去掉  关键字冲突 可加

#查询常量值
SELECT 100;
SELECT 'john';  // SQL不区分字符和字符串就用单引号

#查询表达式
SELECT 100%98;

#查询函数
SELECT VERSION(); // 查询函数的返回值

#起别名
1.便于理解
2.如果要查询的字段有重名的情况，使用别名可以区分
SELECT 100%98 AS 结果;
SELECT last_name AS 名 FROM employees;

SELECT last_name 姓,first_name 名 FROM employees;

SELECT salary as "out put" from employees;

#去重
SELECT DISTINCT department_id FROM employees;

#+号的作用
# 只有运算符的功能chuwu
#只有一方为字符，会试图将字符型转换成数字进行加法运算  没转换成功就是字符0
#一方为null。则结果肯定为null
#查询 员工名和姓连接成一个字段，显示成姓名
SELECT 'last_name' + 'first_name' AS 姓名 FROM employess;错误

SELECT 
  CONCAT(last_name, first_name) AS 姓名 
FROM
  employees ; #正确

```
